sptlbx_writetchstn

Touchstone file write

Calling Sequence

sptlbx_writetchstn(filename, freq, data)
sptlbx_writetchstn(filename, freq, data, R0)
sptlbx_writetchstn(filename, freq, data, R0, comment)

Parameters

filename

string

freq

real-valued row vector

data

square hypermatrix of complex values

R0

real value

comment

strings row vector

Description

Write S-parameter data to Touchstone file

String filename is the filename (excluding extension) of the Touchstone file

Vector freq contains the frequency points of the S-parameters

Hypermatrix data is a square matrix and must be the same depth as the frequency vector freq

Real R0 is the port impedance

Vector comment containts strings placed as comments in the header of the touchstone file prior to the options line. Each element in the string vector is placed as a new comment line

Examples

sptlbx_writetchstn(filename, frequency, data, R0, comment)

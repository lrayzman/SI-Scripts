Name

sptlbx_readtchstn — Touchstone file read

Calling Sequence

[a,b] = sptlbx_readtchstn(c)

Parameters

a

real-valued row vector

b

square hypermatrix of complex values

c

string

Description

Reads S-parameters from Touchstone file referenced by filename string c. Returns frequency points in a and s-parameter data into b.

b is a square complex hypermatrix of the same depth as the frequency vector

Examples

[frequency, data]=sptlbx_readtchstn(filename)

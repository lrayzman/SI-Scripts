sptlbx_writetchstn

Touchstone file write

Calling Sequence

sptlbx_writetchstn(a, b, c)

Parameters

a

string

b

real-valued row vector

c

square hypermatrix of complex values

Description

Write S-parameter data to Touchstone file

String a is the filename (excluding extension) of the Touchstone file.

Vector b contains the frequency points of the S-parameters

Hypermatrix c is a square matrix and must be the same depth as the frequency vector b

Examples

sptlbx_writetchstn(filename, frequency, data)
